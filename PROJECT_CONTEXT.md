# Project Context

This repository is a premium portfolio website for:

Dipta Brata Das  
Director, Sales Engineering & Alliances — AppsCode Inc.

## Tech Stack

Next.js App Router  
React  
TypeScript  
Tailwind CSS  

No UI frameworks should be installed.

Allowed libraries:
- Framer Motion
- Lucide React

## Architecture

src
  app        → Next.js routing
  components → reusable UI components
  sections   → homepage sections
  data       → structured content
  styles     → additional styling
  utils      → helper functions

## Design Philosophy

The site should feel:

- calm
- premium
- technical
- structured
- enterprise-focused

It should resemble a **modern SaaS product site**, not a developer portfolio.

## Development Rules

- Do not install unnecessary libraries
- Prefer small modular components
- Keep section components under ~150 lines
- Reuse UI primitives
- Avoid flashy animations
- Keep interactions subtle and premium
